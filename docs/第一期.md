## css reset

CSS Reset 是每个站点都需要的，最有名的应该是 Eric Meyer 的 Reset CSS，但这也是 10 年前的版本了。作者整理了他的 CSS Reset 版本，包含 9 条规则，进而做了详细解释，说明每一条加的原因和他的考虑。

此外作者其它css文章非常的赞，建议有时间看一下
