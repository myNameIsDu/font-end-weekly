## React Router v6.4

大量新功能

- 使用 `createMemoryRouter/createBrowserRouter/createHashRouter` + `RouterProvider`  创建 路由
- `Route` 的 loader 加载数据 和 action 改变数据
- `errorElement` 处理错误
- `Form` 组件提交数据
- `defer` 和 `Await` 延迟非关键性数据
